# Unreal Template

The full Unreal Engine template project is intentionally not committed here by default.

Recommended distribution options:

1. Attach a zipped template project to a GitHub Release.
2. Host the template on external storage and link it from this file.
3. Commit only a minimal UE template when the required `.uasset` and `.umap` files are small enough, preferably with Git LFS enabled.
